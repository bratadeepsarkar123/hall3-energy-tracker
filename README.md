# Hall-3 Energy Tracker

A zero-dependency, offline-capable HTML app for IIT Kanpur Hall-3 residents to track meals, prevent post-meal sleepiness, and find personal energy patterns.

## Features
- 📅 **Today tab**: Auto-detects day + meal slot. Shows dishes with 🟢🟡🔴 crash risk ratings. Real-time macro calculator.
- 📊 **This Week**: 7-day sleepiness score grid. Click any day to see full menu.
- 📝 **Log Entry**: Log meal, portion size, water intake, post-meal activity, sleepiness 1–10.
- 📈 **Charts**: Scatter plots — sleepiness vs carbs, sleepiness vs protein, activity impact bar chart, daily trend line.
- 🔥 **Live Counter**: Always-available paneer bhurji, omelette, boiled eggs.

## Usage
1. Open `hall3_energy_tracker.html` in any browser
2. Data saves to `localStorage` — persists across sessions
3. Works fully offline once loaded

## Menu Coverage
Full April 2026 Hall-3 rotating menu — 121 dishes across all 7 days:
- Daily base items (bread, sprouts, eggs, roti, rice, dal, sambhar)
- Rotating Breakfast / Lunch / Lunch Extras / Dinner / Dinner Extras per day
- Live counter: Paneer Bhurji, Omelette, Boiled Eggs (always available)
